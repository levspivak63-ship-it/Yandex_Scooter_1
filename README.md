Состав проекта:
1. Проектирование тестов и тестирование веб-приложения:
   - Составлен чек-лист по требованиям к экрану «Статус заказа».Результаты внесены во вкладку «Стр.1 Чек-лист "Статус заказа"».
   - Для экрана «Сделать заказ» составлены проверки на валидацию полей. Результаты внесены во вкладку «Стр.2 Валидация полей "Статус заказа"»
   - Проведено тестирование всей функциональности по макетам и требованиям. Результаты внесены во вкладку «Стр.3 Баги вне тестовой документации»
2. Проектирование тестов для мобильного приложения:
   - Спроектированы тест-кейсы на нотификацию и реакцию приложенияя при отсутствии  интернет-соединения. Результаты внесены во вкладку «Стр.4 Тест-кейсы моб.версии»
3. Проведено ручное тестирование. Выявленные баги внесены в  во вкладку "Баг-репорты"

Тестирование проводилось в средах: 
- Яндекс.Браузер Версия 25.8.5.983, Разрешение экрана 1920 x 1080 ;
- Google Chrome Версия 142.0.7444.135 Разрешение экрана 1280 х 720.

В процессе тестирования обнаружено 40 багов, в том числе: 3 блокирующих;
                                                          14 критических;
                                                          13 стандартных;
                                                          10 желательных.
К баг-репортам приложены скриншеты страниц с указанием ошибок.
